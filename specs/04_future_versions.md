# Future Ideas and Roadmap

This document lists ideas intentionally excluded from MVP.

---

## Observed telemetry

Future versions could automatically capture:

- tool call counts
- execution time
- file changes
- runtime metrics

---

## Model benchmarking

Potential analytics:

- success rate by model
- interruption rate
- confidence calibration
- estimated vs actual effort

---

## Task lifecycle events

Future schemas may include:

- task_start
- task_finish
- task_update

This would enable runtime measurement.

---

## Advanced dashboards

Possible improvements:

- time filtering
- tag exploration
- model comparison
- taxonomy analysis

---

## Taxonomy evolution

Custom tags could be promoted into official vocabularies based on usage frequency.

---

## Distributed telemetry

Future versions could support:

- central telemetry servers
- multi-machine aggregation
- shared datasets

---

## Agent SDK

Possible SDKs:

- Python
- Node
- Go
